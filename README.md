# GoSchedule

GoSchedule is a task scheduler for Go functions.

# Features

- Priority scheduling
- Task dependencies

## Planned Features

- Time-based scheduling
- Repeating execution
- Failed task re-attempt

# Installation

```
go get github.com/williamfedele/goschedule
```
